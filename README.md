# Student Grade Calculator

A simple Python project that calculates the total marks, average marks, and pass/fail status of a student.

## Features

- Accepts marks for 3 subjects
- Calculates total marks
- Calculates average marks
- Displays pass/fail result

## Technologies Used

- Python

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run the following command:

```bash
python grade_calculator.py
```

## Example

Input:

```text
Student Name: Pavani
Subject 1: 80
Subject 2: 75
Subject 3: 90
```

Output:

```text
Total Marks: 245
Average Marks: 81.67
Result: Pass
```
